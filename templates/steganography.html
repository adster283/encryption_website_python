About this tool:


Steganography is the art of hiding text or even files in images. Spies have been using this technique of cryptography for centuries to hide their messages in plain sight. The image while containing all this extra information looks no different to the naked eye when compared with the original image.


This tool is super easy to use, just upload an image along with the text you like to hide within it. You can likewise upload an image with a hidden message and the site will return to you the message it is hiding. Please note that the implementation that this site is using is easy to detect and decrypt by anyone who has the image, so I recommend only using it for educational purposes or sending a fun message to a mate.
